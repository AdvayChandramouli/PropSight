# PropSight
A machine learning pipeline for predicting real estate prices or rental yields using messy, real-world housing data, particularly during tourism spikes --> include some aggregation of data ... Includes end-to-end cleaning, modeling, and deployment with containerized infrastructure.	"Python (Pandas, NumPy, Scikit-Learn, XGBoost), Flask or FastAPI, Docker, Kubernetes, GitHub Actions, optional React for dashboard, PySpark"
